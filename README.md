# python-dpi-network-analyzer
This project is a simple Deep Packet Inspection (DPI) tool built using Python.

It captures network packets and analyzes them to understand how network traffic works.

Features
Capture live network packets
Inspect HTTP traffic
Learn how packet sniffing works
Beginner cybersecurity project
Technologies Used
Python
Scapy
Networking concepts
Installation
Install requirements:

pip install -r requirements.txt

Run the tool
python dpi_sniffer.py## Program Output

Educational Purpose
This project is created for learning cybersecurity and networking.
